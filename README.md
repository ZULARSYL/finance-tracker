# 💵 FINANCE TRACKER
sebuah aplikasi berbasis web untuk mencatat pemasukan, pengeluaran dan mengelola keuangan secara real-time. Dibuat dengan tech Node.js, Express dan MongoDB.

---

## 📌 Fitur Utama

-  Autentikasi Pengguna : Register, Login dan Logout menggunakan Session
- Manajemen Transaksi : Tambah, Edit dan Hapus data pemasukan (Income) dan pengeluaran (Expense)
- Budgeting Pengeluaran Per Bulan : Berisi budget biaya yang perlu di keluarkan perbulan untuk mencapai finansial stabil
- Laporan: Berisi laporan perbulan untuk mengetahui performa biaya pengeluaran dan pemetaan biaya kebutuhan lainnya
- Export to .PDF : mengekspor data keuangan menjadi pdf yang dapat digunakan untuk melihat biaya perbulannya
- Notifikasi Hutang : sebagai pengingat akan pembayaran hutang yang di set di menu notifikasi

---

## Tech Stack

- Backend : Node.js dan Express.js
- Database : MongoDB
- Frontend : Ejs, Bootstrap, Html dan CSS
- Export .PDF : Puppetear
